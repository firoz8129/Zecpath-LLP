# Zecpath AI System

## Project Overview
This project is an AI-powered hiring system that automates resume screening, candidate evaluation, and hiring decisions.

---

## Project Structure

- data/
  Contains sample datasets and resumes

- parsers/
  Handles resume parsing and data extraction

- ats_engine/
  Calculates ATS score for candidates

- screening_ai/
  Handles initial screening logic

- interview_ai/
  Manages interview-related processes

- scoring/
  Final decision logic (Select / Reject)

- utils/
  Helper functions and logging system

- tests/
  Testing files for system validation

---

## Logging & Testing

- Logs are stored in logs/ folder
- Tests are available in tests/ folder

---

## Purpose

To build a scalable AI-based hiring platform with modular architecture.
